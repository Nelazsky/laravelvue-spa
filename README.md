# Laravel Vue SPA 
## Nova Admin panel && Russian language pack

Необходимые версии:
- PHP: 7.4
- Node: 12

## Установка для разработки

- `composer install`
- Добавить информацию о базе данных в `.env`
- `php artisan key:generate`
- `php artisan jwt:secret`
- `php artisan migrate`
- `npm install`
- `npm run dev`
